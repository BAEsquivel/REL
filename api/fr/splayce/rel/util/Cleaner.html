Cleaner

case class Cleaner (clean: (String) ⇒ String, cleanTrack: (TrackString) ⇒ TrackString) extends (String) ⇒ String with Product with Serializable

Cleaner: composable String => String function. Its aim is to clean a String before parsing.

It may have a TrackString => TrackString function to track position shifts when using TrackStrings. Its default implementation is to track nothing when the resulting String keeps the same length, else register a single whole-String replacement.
